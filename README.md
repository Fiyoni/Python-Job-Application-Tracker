# Job Application Tracker (Python)

## Description
A console-based Python application to track and manage job applications. 
Users can add, view, update, and delete job application records such as company name, job role, application date, and status. 
The project uses file handling to store data persistently and is designed to strengthen Python basics and real-world logic.

## Features
- Add new job applications
- View all applied jobs
- Update application status (Applied, Interview, Selected, Rejected)
- Delete job application records
- Persistent data storage using file handling

## Language & Tools
- Python
- File Handling (Text file)

## Skills Learned
- Python fundamentals: variables, loops, conditionals, functions
- File handling for data storage and retrieval
- Lists and dictionaries for structured data
- Menu-driven program design
- Logical thinking and real-world problem solving

- Beginner-friendly Python project
